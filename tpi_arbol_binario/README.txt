“Árbol Binario: Construcción balanceada y búsquedas”
Descripción

Este proyecto consiste en un programa escrito en Python que permite:

Ingresar entre 5 y 9 números uno por uno desde el teclado.
Construir un árbol binario de búsqueda balanceado a partir de esos números.
Mostrar el árbol de forma visual en la consola.
Permitir al usuario buscar un valor dentro del árbol e informar si fue encontrado y cuántos pasos tomó encontrarlo.
El objetivo es aplicar conceptos fundamentales de programación y estructuras de datos, utilizando listas anidadas y funciones recursivas.

Contenidos del programa

construir_arbol(lista)
Crea un árbol binario balanceado a partir de una lista ordenada.
mostrar_arbol(arbol)
Muestra el árbol en consola de forma jerárquica.
buscar_en_arbol(arbol, valor)
Realiza una búsqueda binaria dentro del árbol y cuenta los pasos necesarios.
Cómo ejecutar

Asegurate de tener Python 3 instalado.
Ejecutá el archivo en un entorno como IDLE, Visual Studio Code o directamente desde la terminal con:
python nombre_del_archivo.py
Ingresá los números solicitados por consola.
Observá cómo se crea el árbol y realizá búsquedas sobre él.

¿Qué se aprende con este proyecto?
Estructuras de datos: árboles binarios.
Uso de listas anidadas para representar nodos.
Recursividad en funciones.
Interacción básica con el usuario.
Validación de entradas y control de errores.

Posibles mejoras
Implementar los recorridos preorden, inorden y postorden.
Agregar opción para eliminar nodos del árbol.
Crear una interfaz gráfica para visualizar mejor el árbol.
Guardar e importar los datos ingresados desde un archivo.
